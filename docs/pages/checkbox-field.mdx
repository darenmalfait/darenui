# CheckboxField

CheckboxField is an [Checkbox](/checkbox) with a label, description, and error
message.

<Code code={`import { CheckboxField } from '@daren/ui-components'`} />

## Preview

<Preview code={`<CheckboxField name="preview-field" label="label" />`}>
  <div className="space-y-2">
    <CheckboxField name="preview-field" label="label" />
  </div>
</Preview>

---

## Properties

<Properties>
  <Property name="name" type="string" required>
    The identifier of the checkbox.
  </Property>
  <Property name="label" type="string">
    The label of the checkbox.
  </Property>
  <Property name="error" type="string">
    The error message of the checkbox.
  </Property>
</Properties>
