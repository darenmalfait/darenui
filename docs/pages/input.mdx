# Input

Input is a form component that allows users to enter data. [Field](/field) is an
input field with label, desscription and error fields.

<Properties>

    Whether the input has an error.
  </Property>
  <Property name="icon" type="React.ReactElement">
    An icon to be displayed on the right side of the input.
  </Property>
  <Property name="inputSize" type={['sm' | 'md' | 'lg']}>
    The size of the input.
