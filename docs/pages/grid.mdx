# Grid

Grid is a layout component that allows you to use grid functionality in your
application with some predefined styling.

<Code code={`import { Grid } from '@daren/ui-components'`} />

<Preview direction="vertical">

  </Grid>
</Preview>

## Properties

<Properties>
  <Property name="nested" type="boolean">
    If true, the grid will take up the full width of the parent container.
  </Property>
  <Property name="children" type="React.ReactNode" required>
    The content of the grid.
  </Property>
</Properties>
