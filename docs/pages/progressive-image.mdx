# ProgressiveImage

PrograssiveImage is an image component that loads the image progressively with a
placeholder.

<Code code={`import { ProgressiveImage } from '@daren/ui-components'`} />

---

## Preview

<Preview>
  <ProgressiveImage
    className="w-full"
    img={<img src="https://picsum.photos/3000/3000" alt="picsum" />}
    placeholder="https://picsum.photos/25/25"
  />
</Preview>

---

## Properties

<Properties>
  <Property name="placeholder" type="string">
    The placeholder image url.
  </Property>
  <Property name="img" type="React.ReactNode">
    A node that will be rendered as the image.
  </Property>
</Properties>
